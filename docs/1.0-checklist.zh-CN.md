# 1.0.0 发布前验收清单

[English](1.0-checklist.md) | 简体中文

`1.0.0` 是 Flask-Nacos 的首个稳定版，面向 PyPI 发布。公开 API 在 1.0 系列中保持稳定。
本清单汇总了发布 `1.0.0` 之前必须验证的所有事项。

另请参阅：[发布指南](release.zh-CN.md) - [兼容性](compatibility.zh-CN.md) -
[API 参考](api-reference.zh-CN.md)。

## 验收清单

1. API 已冻结（`scripts/check_api_snapshot.py` 通过）。
2. 版本号已更新为 `1.0.0`（pyproject 与 `__version__`）。
3. `CHANGELOG.md` 已新增 `1.0.0`。
4. `README.md` 已完成最终检查。
5. `docs/` 已完成最终检查（每篇文档都有中英文两个版本）。
6. `examples/` 已完成最终检查。
7. `pyproject.toml` 元数据已完成最终检查（名称、版本、readme、requires-python、
   依赖、dev 额外依赖、分类器、许可证、URL）。
8. `py.typed` 已包含在包中。
9. 没有敏感信息。
10. 没有内部 IP。
11. 没有内部域名。
12. 不提供 `get_config_as_dict`。
13. 不提供 `load_config_to_flask`。
14. 未实现 YAML、JSON、dict 配置解析。
15. `get_config()` 只返回配置原始内容（不做解析）。
16. `pytest` 通过。
17. `ruff check .` 通过。
18. `mypy flask_nacos` 通过。
19. `scripts/check_version.py` 通过。
20. `scripts/check_sensitive_info.py` 通过。
21. `scripts/check_docs.py` 通过。
22. `scripts/check_compatibility.py` 通过。
23. `scripts/check_api_snapshot.py` 通过。
24. `scripts/check_examples.py` 通过。
25. `python -m build` 通过。
26. `twine check dist/*` 通过。
27. `scripts/check_package.py` 通过。
28. `scripts/smoke_test_package.py` 通过。
29. 已完成 TestPyPI 发布与安装验证。
30. 已准备好正式 PyPI 发布。
31. 已准备好 `v1.0.0` Git tag。
32. 已准备好 GitHub Release Notes。

## 如何执行完整检查

```bash
bash scripts/release_check.sh
```

它会运行上述所有检查（不含真实的 TestPyPI/PyPI 上传，上传是
[发布指南](release.zh-CN.md)中描述的独立且需显式执行的步骤）。
